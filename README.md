# LunaSVG [ARCHIVED]

lunasvg is SVG rendering module for Godot Engine. Now in the progress of making apr.

## Features

- Basic Shapes
- Document Structures
- Coordinate Systems, Transformations and Units
- SolidColors
- Gradients
- Patterns
- Masks
- ClipPaths
- Markers
