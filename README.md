# LunaSVG

lunasvg is SVG rendering module for Godot Engine

![svg2png generated PNG](luna.png)

## Features

- Basic Shapes
- Document Structures
- Coordinate Systems, Transformations and Units
- SolidColors
- Gradients
- Patterns
- Masks
- ClipPaths
- Markers
